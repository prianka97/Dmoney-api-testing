# Dmoney-api-testing

## How to run
- Clone this project using `https://github.com/prianka97/Dmoney-api-testing.git`
- Run the below commands
    1. `npm i`
    2. `node report.js`
       
## Prerequisite
- Install Node (latest version)
- Make a directory Name Reports inside the project folder

##Technology 
- Newman
- Postman
